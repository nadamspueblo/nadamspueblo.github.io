Hello!

Welcome to the Pueblo High School Software and App Design (aka Computer Science) program. The mission of this program is to promote, prepare, and inspire students to create and innovate with software and tech.

According to Pipeline AZ, cyber and tech are the fastest growing career fields in Arizona. However, this program isn't soley about career preparation. It's about creativity and invention. It's about gaining access to cutting edge skills, giving yourself a voice, and using that to help solve the problems of the next generation.

Software and App Design CTE Program

Software and App Design is a 2-year Career and Technical Education (CTE) program with optional 3rd and 4th years. Throughout the program, you will have opportunities to earn industry certifications in JavaScript, HTML/CSS, and Java that can be used to gain employment or apply for post-secondary education programs. To complete the program, you must pass the Technical Skills Assessment (TSA).
