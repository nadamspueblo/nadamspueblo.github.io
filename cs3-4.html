CS3-4 is AP Computer Science Principles. It is still an introductory course, but with a more academic focus culminating in the AP exam in the spring semester.

In the fall semester of AP CSP we study how data is represented using binary digits (bits), transmiting binary data over a network, and continue learning Javascript.

In the spring semester, we study algorithms in Javascript for working with strings and lists. Then we prepare to take the state technical skills assessment (TSA) and the AP exam which consists of a multiple choice and project called the AP Create Task that will developed in class.
