scope - where in a script a variable exists
global scope - exist everywhere in a script
local scope - exist only in the function in which declared
conditional block - syntax to execute different blocks of code based on a conditions being true or false
condition - a Boolean value or "test" used in a conditional block
Boolean - a value that is either true or false
comparison operators - operators that are used to compare (test) two values and result in a Boolean value: <, > <=, >=, ==, !=
logical operator - operators that are used to combine Boolean values: &&, ||, and !

Facilitate Classroom Instruction
  • Instruct and explain a specific topic, ask questions
  • Demonstrate code, ask questions
  • Assign practice coding exercises or written questions
  • Circulate to observe and help students as they work
  • Take notes, ask and respond to questions, discuss with partner
  • Copy code, ask and respond to questions, discuss with partner
  • Work to complete exercises or answer questions individually or in pairs
Students discuss with a partner after key information is presented. Students answer questions orally by volunteer and selected randomly by the teacher. Teacher will respond to answers and partner-discussion as necessary to correct misunderstandings.
Students will submit coding assignments and answer a final, summative question at the end of each class period. Teacher will analyze assignments and responses to address issues the following class period as necessary.
Facilitate Lab Instruction
  • Assign lab and present overview and examples
  • Circulate to observe and help students
  • Read instructions and grading rubric
  • Work to complete lab individually or in pairs
Observe and question students as they work. Provide individual feedback as necessary. As common misunderstandings are identified, provide clarification/instruction to the whole class. Plan bell work and closure activities to address common/repeated/important misunderstandings, mistakes, concepts, and skills
