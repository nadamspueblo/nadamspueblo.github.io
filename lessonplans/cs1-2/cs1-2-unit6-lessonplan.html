HTML - Hypertext Markup Language, a language used to create web pages
HTML Element - A piece of a website, marked by a start tag and often closed with an end tag
HTML Tag - The special set of characters that indicates the start and end of an HTML element and that element's type
HTML Attribute - The name of a property that can be set for a particular HTML tag such as the id, src for an image or href for a hyperlink
Website Content - the text and images on a website
Website Structure - how the content of a website is organized
Heading - A title or summary for a document or section of a document.
List - allows web developers to group a set of related items
Copyright - the exclusive legal right to print, publish, perform, film, or record literary, artistic, or musical material, and to authorize others to do the same
CSS - Cascading Style Sheets; a language used to describe how HTML elements should be styled
CSS Selector - the part of a CSS ruleset that defines which HTML elements the style should be applied to
CSS Class - An identifier that allows multiple elements in an HTML document to be styled in the same way
Relevant - closely connected to a topic
Search Engine - A program that searches for items on the World Wide Web.
Trustworthy - reliable, honest, and truthful

Facilitate Classroom Instruction
  • Instruct and explain a specific topic, ask questions
  • Demonstrate code, ask questions
  • Assign practice coding exercises or written questions
  • Circulate to observe and help students as they work
  • Take notes, ask and respond to questions, discuss with partner
  • Copy code, ask and respond to questions, discuss with partner
  • Work to complete exercises or answer questions individually or in pairs
Students discuss with a partner after key information is presented. Students answer questions orally by volunteer and selected randomly by the teacher. Teacher will respond to answers and partner-discussion as necessary to correct misunderstandings.
Students will submit coding assignments and answer a final, summative question at the end of each class period. Teacher will analyze assignments and responses to address issues the following class period as necessary.
Facilitate Lab Instruction
  • Assign lab and present overview and examples
  • Circulate to observe and help students
  • Read instructions and grading rubric
  • Work to complete lab individually or in pairs
Observe and questions students as they work. Provide individual feedback as necessary. As common misunderstandings are identified, provide clarification/instruction to the whole class. Plan bell work and closure activities to address common/repeated/important misunderstandings, mistakes, concepts, and skills
